# Employee Salary Prediction System

A machine learning project that predicts employee salaries using job related features such as experience, company size, technical skills, and role information.

## Project Overview

This project analyzes salary data and builds regression models to estimate employee salaries. It includes data preprocessing, feature engineering, model training, model evaluation, and prediction.

## Features

- Data cleaning and preprocessing
- Missing and invalid value handling
- Feature engineering
- One hot encoding
- Regression model training
- Model comparison
- Salary prediction

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit learn
- Jupyter Notebook

## Models Used

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

## Model Evaluation

The models were evaluated using:

- Mean Absolute Error, MAE
- Root Mean Square Error, RMSE
- R squared Score, R²

Random Forest gave the best performance compared to the other models.

## Workflow

Dataset → Preprocessing → Feature Engineering → Model Training → Evaluation → Prediction
