Employee Salary Prediction System

A Machine Learning based salary prediction project developed using Python and Scikit learn. The system predicts employee salaries based on factors such as experience, company size, technical skills, and job related information.

Project Overview

The goal of this project is to analyze real world salary datasets and build regression models capable of predicting salaries accurately. The project focuses on data preprocessing, feature engineering, model training, and performance evaluation.

Features
Data cleaning and preprocessing
Handling missing and invalid values
Feature engineering
One hot encoding for categorical data
Multiple regression models
Model evaluation and comparison

Dataset Processing

The dataset was cleaned by:

Removing unnecessary columns
Handling missing values
Converting categorical values into numerical format
Encoding company size and job related features
Model Evaluation

The models were evaluated using:

Mean Absolute Error (MAE)
Root Mean Square Error (RMSE)
R squared Score (R²)

Random Forest achieved the best performance with higher prediction accuracy compared to other models.

Dataset → Preprocessing → Feature Engineering → Model Training → Evaluation → Prediction
